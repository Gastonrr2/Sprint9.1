¿Qué es Node JS?
Node.js es un entorno de ejecución de JavaScript de código abierto que se ejecuta en el servidor. 
Fue creado para desarrollar aplicaciones de red escalables y de alto rendimiento.

¿Es Node JS un lenguaje de programación?
No, Node.js no es un lenguaje de programación. Es un entorno de ejecución para JavaScript construido sobre el motor JavaScript V8 de Chrome.

¿Puedo tener acceso al DOM desde Node JS?, ¿por qué?
No, no puedes acceder directamente al DOM desde Node.js. El DOM es una API para documentos HTML y XML y es una 
característica de los navegadores web, no de JavaScript en sí. Node.js se ejecuta en el servidor, no en el navegador, por lo que no tiene acceso al DOM.

¿Qué diferencia existe en instalar una dependencia agregando –dev?
Se instalan los módulos que solo se necesitan durante el desarrollo y el proceso de construcción. No se instalan en producción.

¿Qué es Express?
Express es un marco de aplicación web para Node.js. Fue diseñado para construir aplicaciones web y API de manera rápida y fácil. Es minimalista, 
flexible y proporciona un conjunto robusto de características para las aplicaciones web y móviles.

¿Hay algún otro framework en NodeJs que hace algo similar a Express?

Sí, existen  otros marcos de trabajo en Node.js que ofrecen funcionalidades similares a Express.js.
Por ejemplo: 
-Koa.js
-Hapi.js
-Fastify
-NestJS.

¿Por qué express frente a otros frameworks?
Express.js es popular debido a su simplicidad, flexibilidad, documentación completa, gran comunidad de desarrolladores, 
madurez y buena integración con otras tecnologías. Es minimalista, no impone una estructura de proyecto específica, y 
permite una gran personalización con plugins y middleware. Sin embargo, la elección del framework debe basarse en las 
necesidades específicas de tu proyecto.
